### ✅ 핵심기능
- [ ] 메인화면 : 다른 사람이 쓴 리뷰 보기(본인 리뷰를 제외한 최신순 6개)
- [ ] 회원가입/로그인 : 이메일 인증을 통한 회원가입 진행
- [ ] 검색 : 카카오 책 api 를 통한 검색 기능
- [ ] 리뷰작성 : 이모지를 사용한 책 리뷰 작성기능
- [ ] 통계 및 리뷰 모아보기 : 작성한 리뷰를 기반으로 한 통계 및 리뷰 모아보기 기능
- [ ] 개인설정 페이지 : 비밀번호, 닉네임 변경, 회원탈퇴, 로그아웃 기능
